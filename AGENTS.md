# Project Goal

Build a modern, trustworthy, responsive physiotherapy clinic website.

## Design Direction

- Clean medical + wellness design
- White / light background
- Soft blue, teal, mint, and warm accent colors
- Calm, professional, premium look
- Rounded cards, generous spacing, soft shadows
- Mobile-first responsive design
- Avoid clutter

## Website Sections

- Hero section
- Services
- About clinic
- Why choose us
- Treatment process
- Testimonials
- Appointment CTA
- Contact section
- Footer

## Rules

- Use semantic HTML
- Keep CSS clean and readable
- Make it responsive for mobile, tablet, desktop
- Use accessible contrast
- Add subtle animations only where useful
- Do not overuse gradients
- Use placeholder image areas if no real images are provided

## Done Means

- Website looks polished
- Works on mobile
- Header/navigation works
- CTA buttons are visible
- Layout does not break
